MacOS 添加开机启动命令

2019-05-19 1 min read # macos

😳 准备好要执行的命令

创建可执行脚本文件

> vi ~/frp/frpc.sh

按格式输入命令

#!/bin/bash

# 启动客户端放入后台执行
~/frp/frpc -c ~/frp/frpc.ini &
exit

修改完后保存退出

:wq

添加可执行权限

> chmod 755 ~/frp/frpc.sh

添加开机启动

打开 系统偏好设置 应用，进入 用户与群组 -> 登录项，点击 + 加号，选择刚刚创建的文件

开机后如果显示 “进程已结束” 的终端，可进入 终端 的 偏好设置 -> 描述文件 -> 默认描述 -> Shell 将 当shell退出时： 选为 关闭窗口
