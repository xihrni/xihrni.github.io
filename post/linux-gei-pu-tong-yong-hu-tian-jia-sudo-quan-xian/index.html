Linux 给普通用户添加 sudo 权限

2016-04-19 1 min read # linux

😳在 Linux 中，普通用户可以通过添加到 sudo 组来获得 sudo 权限

  1. 首先使用管理员账户登录
  2. 使用以下命令打开 /etc/sudoers 文件：
sudo nano /etc/sudoers
  1. 在文件中查找以下行：
%sudo ALL=(ALL:ALL) ALL
  1. 添加普通用户到 sudo 组（其中，username 是普通用户的用户名）：
sudo usermod -aG sudo username
  1. 保存文件并退出：
Ctrl + O
Ctrl + X
  1. 让普通用户登录系统，并使用 sudo 命令执行需要管理员权限的操作。

这样，就已经成功地给普通用户添加了 sudo 权限。注意，使用 sudo 命令执行的操作是有风险的，因此请谨慎操作。
