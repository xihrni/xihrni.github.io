PHP、Nginx、云存储跨域配置

2018-09-02 1 min read # 跨域 # 腾讯云 # 阿里云 # php

😳前端调用接口或者JS静态文件浏览器都会出现跨域问题，禁止访问。所以需要在后端提供请求的时候带上允许跨域的提示

PHP

$origin = $_SERVER['HTTP_ORIGIN'] ?? '';
if (in_array($origin, ['https://www.xihrni.com'])) {
    // 允许来源（可使用*代替所有）
    header('Access-Control-Allow-Origin: ' . $origin);
    // 允许请求方式（可使用*代替所有）
    header('Access-Control-Allow-Methods: GET, POST, PUT, PATCH, DELETE, OPTIONS');
    // 允许携带Cookie
    header('Access-Control-Allow-Credentials: true');
    // 允许请求头（不能使用*）
    header('Access-Control-Allow-Headers: X-Requested-With, Content-Type, X-CSRF-Token, Token, Accept');
}

Nginx

# vhost/*.conf

server
{
    # 上文
    ...

    # 允许跨域请求
    add_header 'Access-Control-Allow-Origin' *;
    add_header 'Access-Control-Allow-Methods' *;
    add_header 'Access-Control-Allow-Credentials' true;
    add_header 'Access-Control-Allow-Headers' *;

    # 下文
    ...
}
